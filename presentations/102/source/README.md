# Source Notes

The 102 slide source is authored with the Codex Presentations skill using artifact-tool presentation JSX.

- `slides/deck.mjs` contains the shared visual system, slide data, and renderers.
- `slides/slide-01.mjs` through `slides/slide-13.mjs` define individual slides.
- `generate-assets.mjs` records the Grok Imagine prompts used for supporting images.

Slide 13 is a presenter-only appendix cue for the Fidelity Tasks VS Code demo and is intentionally excluded from Ara narration.
